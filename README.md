# PLCT Interview Exam (2025)

## 简介
本项目包含 2020 夏季上机试题的完整 C++ 实现及自动化测试套件。

## 环境要求
- G++ (支持 C++11)
- Python 3 (用于生成测试数据)
- Bash (用于运行测试脚本)

## 文件说明
- `mixplus.cpp`: Q1 16进制/10进制加法器
- `cnt.cpp`: Q2 字符串频率统计
- `check.cpp`: Q3 线段相交检测
- `Makefile`: 自动化编译脚本
- `judge.py`: 测试数据与标准答案生成器
- `duipai.sh`: 自动化黑盒测试/对拍脚本

## 快速开始

### 1. 编译代码
```bash
make all
